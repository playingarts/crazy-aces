# Agent Reports

This folder contains reports from specialist agents. The Team Lead reads these to coordinate work.

## File Naming Convention

| Agent | Report File |
|-------|-------------|
| Architecture | `architecture-report.md` |
| Game Design | `game-design-report.md` |
| Security | `security-report.md` |
| Testing | `testing-report.md` |
| UX | `ux-report.md` |

## How to Use

### For Specialist Agents
After completing your analysis, save your findings:
```
Write your report to: .claude/reports/{agent}-report.md
```

### For Team Lead
Read reports from: `.claude/reports/`

## Report Format

Each report should include:
- **Date**: When the report was generated
- **Summary**: Key findings in 2-3 sentences
- **Details**: Full analysis
- **Recommendations**: Prioritized action items
- **Status**: Open issues / Resolved items
